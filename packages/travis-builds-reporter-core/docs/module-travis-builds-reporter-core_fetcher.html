(inner) fetch(repositoryName, connection) → {Promise.<array>}

Entry point of the fetcher module. This method calls internal methods that:
  1. validate Travis APIs connections and responses
  2. initialize basic informations needed for the asynchonous fetcher function
  3. actually make the concurrent calls to fetch builds
Parameters:
Name Type Description
repositoryName string Name of the repository from which the module will try to fetch builds
connection array Correctly configured axios instance to make HTTP calls (@see module:travis-builds-reporter-core/client)
